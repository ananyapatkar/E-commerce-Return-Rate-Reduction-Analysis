# YouTube Trending Video Analytics

## 📌 Objective
Analyze YouTube trending videos across regions to uncover patterns in views, categories, and sentiment.

## 🛠 Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI

## 📊 Features
- Data cleaning and preprocessing
- Sentiment analysis on video titles
- Category ranking using SQL
- Time-series analysis of trending videos
- Interactive Power BI dashboard

## 📈 Key Insights
- Entertainment categories dominate trending videos
- Positive sentiment titles attract more views
- Engagement varies across regions
- Most videos trend for short duration

## 📁 Project Structure
- data/ → cleaned dataset
- notebooks/ → analysis code
- dashboard/ → Power BI file
