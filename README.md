# E-commerce Return Rate Reduction Analysis

## 📌 Project Overview
Product returns are a major challenge in e-commerce, increasing operational costs and affecting profitability.  
This project focuses on analyzing return patterns and predicting the likelihood of product returns using data analytics and machine learning

---

## 🎯 Objective
- Identify key factors influencing product returns  
- Analyze return rates by category, region, and product  
- Predict return probability using Logistic Regression  
- Build an interactive Power BI dashboard for insights  

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)
- Data Visualization (Matplotlib, Seaborn)
- Machine Learning (Scikit-learn - Logistic Regression)
- Power BI (Dashboard & Data Visualization)

---

## 📊 Key Features
- Data cleaning and preprocessing  
- Return rate analysis by category and region  
- Identification of high-risk products  
- Logistic Regression model for return prediction  
- Interactive Power BI dashboard with filters  

---

## 📈 Dashboard Insights
- Total Orders, Returns, and Return Rate KPIs  
- Return trends by Category and Sub-Category  
- High-return products identification  
- Sales vs Return Risk analysis  
- Impact of shipping mode on returns  

---

## 🤖 Machine Learning Model
- Model Used: Logistic Regression  
- Target Variable: Return_Flag (0 = No Return, 1 = Return)  
- Evaluation Metrics:
  - Accuracy Score  
  - Precision, Recall, F1-score
